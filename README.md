# Basic Ransoware

> **Status:** Em desenvolvimento

## Sobre

Projeto educacional e experimental em Ruby que demonstra conceitos de criptografia de arquivos e manipulação de interface gráfica em tela cheia. **Este repositório está em desenvolvimento** — o código ainda está sendo implementado e testado.

## Objetivo

* Estudar e demonstrar princípios de criptografia simétrica (AES) em Ruby.
* Experimentar interfaces em modo fullscreen com GTK3.
* Construir um protótipo para fins educacionais e de pesquisa em segurança.

## Status do Desenvolvimento

* Implementação inicial da interface gráfica (GTK3): Em desenvolvimento.
* Funções de criptografia: Em desenvolvimento.
* Processamento recursivo de pastas: Planejado.
* Função de descriptografia: Planejada.

## Dependências

O projeto usa bibliotecas Ruby que precisam estar instaladas no ambiente de desenvolvimento, por exemplo: `gtk3`, `openssl`, `bcrypt` e outras bibliotecas padrão do Ruby.
